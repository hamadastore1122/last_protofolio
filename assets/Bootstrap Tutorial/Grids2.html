Tip: Remember that grid columns should add up to twelve for a row. More than that, columns will stack to maintain the viewport.

Grid Classes

The Bootstrap grid system has four classes:

  • .col-xs-* (for phones - screens less than 768px wide)
  • .col-sm-* (for tablets - screens equal to or greater than 768px wide)
  • .col-md-* (for small laptops - screens equal to or greater than 992px wide)
  • .col-lg-* (for laptops and desktops - screens equal to or greater than 1200px wide)

Basic Structure of a Bootstrap Grid

The following is a basic structure of a Bootstrap grid:

  
  
<div class="container">  
    <div class="row">  
        <div class="col"></div>  
        <div class="col"></div>  
    </div>  
    <div class="row">  
        <div class="col"></div>  
        <div class="col"></div>  
    </div>  
</div>

Grid Options

Class prefix Suitable for Grid behaviour Container width # of columns
.col-xs-* Phones Horizontal at all times None (auto) 12
.col-sm-* Tablets Collapsed to start, horizontal above breakpoints 750px 12
.col-md-* Small Laptops Collapsed to start, horizontal above breakpoints 970px 12
.col-lg-* Laptops & Desktops Collapsed to start, horizontal above breakpoints 1170px 12
