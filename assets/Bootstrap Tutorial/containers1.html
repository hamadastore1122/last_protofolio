Bootstrap Containers

Containers

Containers are a fundamental building block of Bootstrap that contain, pad, and align your content within a given device or viewport.

How they work

Containers are the most basic layout element in Bootstrap and are required when using our default grid system. Containers are used to contain, pad, and (sometimes) center the content within them.

Default container

<div class="container">
<!-- Content here -->
</div>

Responsive containers

Responsive containers allow you to specify a class that is 100% wide until the specified breakpoint is reached.

Fluid containers

<div class="container-fluid">
<!-- Content here -->
</div>

CSS

Sass variables and mixins are available to customize your layouts.
