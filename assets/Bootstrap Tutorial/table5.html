Responsive Tables

Add the class .table-responsive to create a responsive table that adds a scrollbar when the table is too wide horizontally:

Example

# First Name Last Name Age City Country Gender Example 1 Example 2 Example 3 Example 4 Example 5
1 Anna Smith 35 New York USA Female Yes Yes Yes Yes Yes

You can also specify when to show the scrollbar based on the screen width:

  • .table-responsive-sm - for screens less than 576px
  • .table-responsive-md - for screens less than 768px
  • .table-responsive-lg - for screens less than 992px
