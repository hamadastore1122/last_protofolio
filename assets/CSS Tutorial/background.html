CSS Backgrounds

By mohammed yahya

Background properties are used to apply certain effects to the background of the design in general.

Background Color

The background-color property is used to specify the background color of a specific element in a web page. The background color of a web page is specified by applying the background color to the body selector.

  
body {background-color: #bec4de;}

There are 3 types of color customization using CSS as follows:

  • "#FFFFFF" Using a HEX value example
  • "rgb(255,0,0)" Using RGB value example
  • Using the color name like "Red"

In this example, it is shown that the h1, p, div tags have different background colors:

  
h1 {background-color:#6495ed;}  
p {background-color:#ffffff;}  
div {background-color:#bec4de;}

Background Image

The background-image property is used to specify an image as the background for a specific element. The image is repeated by default until it covers the element it is entered into.

The background image of the web page is specified as in the following example:

  
body {background-image: url('smile.jpg');}

Repeat Background Image – Horizontal or Vertical

According to the default settings, the image will be repeated on both the horizontal and vertical sides.

You must choose the image and location well and determine whether it should be repeated horizontally, vertically, or without repeating, according to the design and location of your element.

If you want to repeat the image horizontally, then you can use the value repeat-x and the background will become as in the following example:

  
body {  
    background-image: url('smile.jpg');  
    background-repeat: repeat-x;  
}

Background Image – Repeat Vertically

If you want to repeat the image vertically, then you can use the repeat-y value and the background will become as in the following example:

  
body {  
    background-image: url('smile.jpg');  
    background-repeat: repeat-y;  
}

Background Image – Set Position and No Repeat

When using an image as a background for the site, use an image that does not hide the text written underneath it. The image can be shown in the background once through the background-repeat property:

  
body {  
    background-image: url('smile.jpg');  
    background-repeat: no-repeat;  
}

The image can hide the text or not show it clearly, so we can modify the position of the image. Here we can use the background-position property:

  
body {  
    background-image: url('smile.jpg');  
    background-repeat: no-repeat;  
    background-position: right top;  
}

Background Property Abbreviation

Note from the previous examples that there are many properties that deal with the background. The code can be abbreviated and collected in one property instead of placing all the properties:

  
body {  
    background: #ffffff url('smile.jpg') no-repeat right top;  
}

Property Order

As you can see, we can collect all the properties into one property, but you must pay attention that the values must be arranged as follows:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

It doesn't matter if one of the values is missing, but the important thing is to keep their order exactly as I mentioned.
