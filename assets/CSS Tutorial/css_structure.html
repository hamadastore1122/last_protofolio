CSS Syntax

To use CSS, there are two primary components: the selector and the declaration.

  
h1 {  
    color: Red;  
    font-size: 12px;  
}

The selector is the HTML element you want to style, like an `h1` or `body`, without using the tags themselves.

The declaration consists of a property and a value.

The property refers to the design aspect you want to change, such as font or color, and each property has a corresponding value.

Example of CSS

  
p {  
    color: red;  
    text-align: center;  
}

Each declaration should end with a semicolon.

Comments in CSS

You can use comments in your stylesheet to explain the code you're using, which can help when modifying your website's design.

  
/* Comment: Now specifying the paragraph tag */  
p {  
    text-align: center;  
    /* Another comment */  
    color: black;  
    font-family: Arial;  
}
