Font Type

By mohammed yahya

The font-type property in CSS is used to specify the font family, size, and style of the text font.

Font Family

A font family is a group of font types used with the font-family property. The font-family property must contain many font types because if the browser does not support the first, it moves to the second, and if it does not support the second, it moves to the third, and so on.

First, use the font you want, and then use the general fonts to allow the browser to use the general fonts if it does not support the fonts you have previously placed.

Note: If the font type name contains more than one word, then it must be placed between quotation marks. Example:

font-family: "Times New Roman";

If you put more than one type of font, then you can separate them with a comma. For example:

p {font-family: "Times New Roman", Times, serif;}

Font Style

The font-style property is widely used to specify the italic of the text. This property has three different values:

  • normal - the text appears normally
  • italic - the text appears in italics
  • oblique - The text appears curved (curved is the same as italic but is not supported by browsers, so it is preferable to use italic).

Example:

p.normal {font-style: normal;}  
p.italic {font-style: italic;}  
p.oblique {font-style: oblique;}

Font Size

The font-size property sets the font size. Being able to set the font size is an important step in the perfect design of your website. But don't use font-size to make the paragraph look as big as the title or the title look as small as the paragraph.

You can always use HTML tags for headings from <h1> to <h6> or <p> for paragraphs. The font size value can be absolute or continuous.

Absolute Size

  • Adjust text to a specific size.
  • User is not allowed to change font size in all browsers (bad move for accessibility reasons).
  • Absolute size is useful especially when the apparent size of the text is known.

Relative Font Size

Adjust the size of connected text around elements.

This allows the user to change the font size in browsers.

If you do not specify a font size, the default text size is normal, like a paragraph, and the size is exactly 16 pixels (1em = 16px).

Adjust Font Size by Pixels

Adjusting the font size by pixels will give you complete control over the size of the text. For example:

h1 {font-size: 40px;}  
h2 {font-size: 30px;}  
p {font-size: 14px;}

In the above example, web browsers such as Internet Explorer 9, Firefox, Chrome, Opera, and Safari will allow adjusting the text size.

Note: The previous example does not work on Internet Explorer versions older than 9.

The text size can be adjusted through all browsers using the zoom tool (the entire page size can be adjusted, not just the text).

Adjust Font Size with em

To avoid annoying problems with older versions of Internet Explorer and others, programmers were able to use the em unit of measurement instead of pixels. The World Wide Web Consortium (W3C) recommends using the em unit of measurement.

Each 1em is equivalent to the normal font size, as the default size in browsers is 16px, so the default size is considered 1em.

The font size can be calculated by dividing the pixels by 16, and the result will appear in em units.

h1 {font-size: 2.5em;} /* 40px/16=2.5em */  
h2 {font-size: 1.875em;} /* 30px/16=1.875em */  
p {font-size: 0.875em;} /* 14px/16=0.875em */

In the previous example, the font size was set in em units, and it matches the size we chose in the previous example using pixels. However, now you can use the font size setting in all browsers, whether old or new, with the em unit of measurement.

Unfortunately, some older versions of Internet Explorer (IE) make the text too large when set to large or too small when set to small.

Use a Combination of % or em

The solution that works on all browsers is to set the default font size by percentage in the <body> tag.

Example:

body {font-size: 100%;}  
h1 {font-size: 2.5em;}  
h2 {font-size: 1.875em;}  
p {font-size: 0.875em;}

Now the code works fine and displays correctly on all browsers.
