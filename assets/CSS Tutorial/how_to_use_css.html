How do I use CSS?

By mohammed yahya

When the browser reads the design page, the HTML page will be displayed according to the custom design.

There are three types of CSS code attachments:

  • External design file
  • Interior design
  • In-line design

External design file

The external file is ideal when applying a specific design to many pages. Through the external file, you can control the design of your site through only one file, but each HTML page must be linked to that external file by using the <link> tag inside the <head> tag while maintaining the same extensions inside the <link> tag such as:

  
<link rel="stylesheet" type="text/css" href="mystyle.css" />

The external design file can be written with a text editor as we mentioned when using HTML. No HTML tags should be used in the design file. It can be saved in an independent external file with the .css extension. The following example shows the CSS codes inside the independent external file:

  
hr {color:sienna;}  
p {margin-left:20px;}  
body {background-image:url("images/back40.gif");}

Note: Do not leave a space between the value property and the unit used, such as "margin-left:20 px" instead of "margin-left:20px", meaning that 20 and px should not have a space between them (20px), because the wrong method will only work in Internet Explorer, but not in Firefox and Opera.

Interior design

Internal design can be used when using a unique design for a specific web page. Then we can use an internal design, i.e., in the same web page, through the <style> tag and using the type="text/css" extension, i.e., the type of design CSS, and it is also used inside <head>.

  
<style type="text/css">  
hr {color:sienna;}  
p {margin-left:20px;}  
body {background-image:url("images/back40.gif");}  
</style>

In-line design

Inline design, i.e., inside an HTML tag, causes loss of professional design features by mixing HTML tags with CSS codes, so use this method economically and wisely.

To use inline design, use the style extension inside the tag you want to modify. You can also use any CSS property and value. The following example shows that this particular paragraph will be applied to the brown color design and a 20 pixel right-spaced page:

  
<p style="color:sienna; margin-right:20px;">This is a brown paragraph</p>

Multiple design pages

If more than one property is set for the same selector in multiple design files, then the value that is most specific to the selector will be used. This example is an external file and the following properties have been selected for the selector h3:

  
h3 {  
    color:red;  
    text-align:left;  
    font-size:8pt;  
}

In the interior design, these properties were chosen for the h3 selector:

  
h3 {  
    text-align:right;  
    font-size:20pt;  
}

If the web page is connected to an external design file and contains an internal design, the following design will appear for the h3 tag:

  
color:red; text-align:right; font-size:20pt;

The color is adopted by the external file and the alignment and font size are replaced by the internal design.

Streamlined arrangement

Which designs will be used when there is more than one design for the same element on a web page?

In general, we can say that all designs will be combined in one functional design file through the following rules, where number 4 has the highest priority:

  1. Default browser
  2. External design file
  3. Interior design file
  4. Inline design (inside HTML tags)

Note: Multiple external design pages can be used for a single HTML page.
