Hyperlinks

By mohammed yahya

Hyperlinks can be designed and modified in many ways.

Edit hyperlinks

CSS hyperlinks can be styled with several properties (color, font family, background, etc.). In particular, you can style and modify the link according to the state it belongs to:

There are four states that a hyperlink falls into:

  • a:link: a natural link that has not been visited or clicked.
  • a:visited: the link that was visited by the user.
  • a:hover: when the user places the mouse on the link without clicking on it.
  • a:active: the moment the link is clicked.
a:link {color:#FF0000;} /* not clicked */  
a:visited {color:#800080;} /* clicked */  
a:hover {color:#FF00FF;} /* when mouse over it */  
a:active {color:#0000FF;} /* when clicked */

When setting the design for all types of links, you should focus on the following conditions:

  • a:hover must be used after a:link and a:visited.
  • a:active must be used after a:hover.

Text Decoration

The text-decoration property is widely used to remove the line under the hyperlink:

a:link {text-decoration:none;}  
a:visited {text-decoration:none;}  
a:hover {text-decoration:underline;}  
a:active {text-decoration:underline;}

Background Color

The background-color property is used to customize the background color of a hyperlink:

a:link {background-color:#B2FF99;}  
a:visited {background-color:#FFFFB5;}  
a:hover {background-color:#FF704D;}  
a:active {background-color:#FF704D;}
