Call a Special Class or ID

In the previous lesson, we talked about how we can use the (selector), which is an HTML tag, to make a complete modification to that tag or selector in all HTML pages. See this example:

  
h1 { color: red; text-align: center; }

This example means that every large heading tag defined by the h1 tag in the web page will be colored red and centered. But what if I have 5 large headings in the web page, this design will be applied to them. What should I do to make 4 headings red and one blue?

Using CSS, you can apply a specific design to a specific tag in the web page by using the class and id calling properties. These two properties enable you to call a specific design with a unique name that you have chosen:

  
h1 { color: red; }  
.heading { color: blue; }

In this example, we mean that all the headings in the web page that have the h1 tag will be red, while the others will be blue. Although heading is not a tag, it is a unique name that I can call with any tag I want, and it must be called through HTML tags as follows:

  

This is a blue title
  

This is a blue paragraph

But the question is which is better, class or id, and when can they be used?

Use ID

The id can be used to apply a specific design to a unique HTML tag. The id is used with only one tag. You must add "#" to the name you want to apply the design to and call it through the id extension in the HTML tag.

  
#para1 {  
    text-align: center;  
    color: red;  
}

Note: When calling the name as a suffix in the tag, just add the name and do not add "#" to the value.

  

This is a heading

Do not start the name when using id with a number because it will not work on Mozilla/Firefox browser.

Use Class

A class can be used to apply to a tag or a group of tags, unlike id which can be applied to only one tag. A class allows you to use a specific style and apply it as many times as needed.

You must add a dot "." to the name you want to apply the design to and call it through the class extension in the HTML tag.

  
.center { text-align: center; }

In the following example, all elements that use the name and the class="center" extension will be center aligned:

  
p.center { text-align: center; }

You can also customize a specific tag that will only have this design applied to it. In the following example, all paragraphs/topics that use the name and the suffix class="center" will have center alignment applied.

Do not start the name when using class with a number because it is only supported by Internet Explorer.
