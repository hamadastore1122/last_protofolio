What is CSS language?

CSS is an abbreviation for Cascading Style Sheets.

Styling is how HTML tags appear.

The design has been added to HTML to solve website design problems.

The design can be saved as an external .css file which in turn saves a lot of work and effort.

External design files are saved in .css files.

Design language can solve big problems

It was not intended from the beginning for the HTML page to form a specific design through HTML tags, but the main work of the HTML page is to describe the content of the page only, such as:

  

This is a great title
  

This is a paragraph

When tags like <font> and color were added as a suffix to HTML 3.2 in particular, designing and modifying websites became nightmares for programmers, especially modifying huge websites where each page contains a separate design from the other. Also, modification took a lot of time and effort.

To solve this problem, the World Wide Web Consortium (W3C) invented the CSS language in HTML 4.0. All tags used to create designs in an HTML page were removed and replaced and saved in a CSS file.

All browsers support CSS today.

CSS saves a lot of work

CSS describes how HTML tags are displayed.

CSS files are saved in an external file with the extension .css. These files allow you to modify the design of all pages on your site in just one file. Imagine!
