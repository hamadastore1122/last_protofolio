Text Color

The color property is used to color a specific text using the CSS language. You can use one of these methods to color a specific text:

  • "Using a HEX value example: #FFFFFF"
  • "Using RGB value example: rgb(255,0,0)"
  • "Using the color name like Red"

The default color of the page is defined by the body selector.

  
body {color:blue;}  
h1 {color:#00ff00;}  
h2 {color:rgb(255,0,0);}

Note: If you specify the color property, then you must also specify the background-color as well.

Text Alignment

The text-align property is used to set the horizontal alignment of text. Text can be centered, right-aligned, left-aligned, or justified. When the justify value is used, each line will be extended and equal in width, and the left and right margins are straight (as in newspapers and magazines).

  
h1 {text-align:center;}  
p.date {text-align:right;}  
p.main {text-align:justify;}

Text Decoration

The text-decoration property is used to set or remove a certain decoration for text. Text-decoration is widely used to remove the line from under the text within a hyperlink for decorative purposes.

  
a {text-decoration:none;}

You can also decorate the text, for example:

  
h1 {text-decoration:overline;}  
h2 {text-decoration:line-through;}  
h3 {text-decoration:underline;}  
h4 {text-decoration:blink;}

Note: It is not preferable to underline any text unless it is a hyperlink because it causes some confusion for the user.

Text Transformation

The text transform feature is used to convert text to lowercase or uppercase. It is used to convert all text to uppercase or lowercase, or to capitalize the first letter of each word.

Note: This feature is only used with words in English, not Arabic.

  
p.uppercase {text-transform:uppercase;}  
p.lowercase {text-transform:lowercase;}  
p.capitalize {text-transform:capitalize;}

Text Indentation

The text spacing property is used to specify the spacing of the first line of a specific paragraph (used with English language paragraphs).

  
p {text-indent:50px;}
