Color Values

HTML colors are specified by the hexadecimal (HEX) system of red, green, and blue (RGB).

The lowest value that can be given to a color is 0 (HEX: 00) and the highest value that can be given to a color is 255 (HEX: FF).

HEX values are specified by three pairs of single digits that begin with the sign #.

The combination of the three colors red, green, and blue from 0 to 255 gives more than 16 million different colors (256 x 256 x 256).
