Forms

The form can contain input elements such as text fields, selecting more than one element (checkboxes), approval buttons, and more.

Forms - Input Elements

Inputs are used to select user information. The input element can be used in many ways depending on the type of input.

Text field

Defines a single-line text input that allows the user to type text.

<form>  
First Name: <input type="text" name="firstname"><br>  
Second name: <input type="text" name="lastname" />  
</form>

Single radio buttons

Define a single radio button. These buttons allow you to select only one option from several others.

<input type="radio" name="sex" value="male" /> Male<br>  
<input type="radio" name="sex" value="female" /> Female<br>

The form code will appear in the browser as follows:

Male Female

Multiple Selection

Defines a multiple selection that allows the user to select one or more of the multiple options available to him.

<input type="checkbox" name="lesson" value="HTML" /> I study HTML<br>  
<input type="checkbox" name="lesson" value="CSS" /> I study CSS<br>

The form code will appear in the browser as follows:

I study HTML I study CSS

Submit button

Specifies the submit button. The submit button allows data to be sent from the form to the server.

<form name="form" action="send_data.php" method="get">

Password: <input type="password" name="pwd" />
