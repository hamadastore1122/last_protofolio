HTML Headings

<HTML headings are defined by tags <h1> through <h6>.

<H1> defines the most important heading.

<H6> defines the least important heading.

:Example

<h1>This is the title</h1>

<h2>This is the title</h2>

<h3>This is the title</h3>

.Browsers automatically add space (margin) before and after each title.

Titles are important

.HTML headings are used for headings only. They are not used to make text big or bold.

.Search engines use headings to index the structure and content of your web pages.

Since users may browse your web pages by their titles, it is important to use headings to show the structure of your document.

H1 headings should be used as the main headings, followed by H2 headings, then less important H3 headings, and so on.

HTML Fonts

.The <hr /> tag creates a horizontal line in an HTML page.

.The hr element can be used to separate content.

:Example

<p>This is a topic</p>

<hr />

<p>This is a topic</p>

<hr />

<p>This is a topic</p>

HTML Comments

.Comments can be added to HTML code to make it more readable and understandable.

.Comments are ignored by the browser and are not displayed.

:Comments are written as follows

<!-- This is a comment -->

.There is an exclamation mark after the opening parenthesis but not before the closing parenthesis.

How to view HTML source

?Have you ever seen a web page and wondered how they did it

To find out, right-click on the page and select "View source" in Internet Explorer.

"View source of this page" in Firefox, or similar in other browsers.

.This will open a window with the HTML code for the page.
