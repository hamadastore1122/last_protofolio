Images - Tag and src Extension

By mohammed yahya

Images are defined in an HTML page by the <img> tag. The <img> tag is an empty tag, meaning it does not contain the tag ending, but only contains extensions inside it.

To display the image on the page, you need the extension src, which means the source of the image, and the value inside src is the link to the image you want to display.

Image Code Structure

<img src="url" alt="some text"/>

URL is used to locate an image, for example, if the image name is boat.jpg and it is located in the images folder on the website www.example.com, then the image URL will be http://www.example.com/images/boat.jpg.

The browser displays the image when the <img> tag is declared on the page. If you put the image tag between paragraphs, the browser will display a paragraph on the page, then the image, then the second paragraph.

Images - Alt Suffix

The alt suffix is used to specify an alternative text or a simple description of the image in case the image is not displayed due to an incorrect link, the image has been deleted from the source, or due to a slow connection. The value inside alt is an alternative text (description) from the author of the image.

<img src="boat.jpg" alt="big boat" />

Images - Adjust the Height and Width of the Image

The height and width of the image can be specified using the height and width extensions, and the values used are specified in pixels, as in the following example:

<img src="home.jpg" alt="The Old House" width="304" height="228" />

Useful Notes

It is important to use the height and width extensions for the image because when using these properties the image will be selected in its designated place, but if the height and width are not specified the image may appear in its normal size and then it can cause problems in the structure of the site.

When the page loads, the browser is the one that fetches the image from the server and displays it on the site, so be careful when using images and do not change the image folder unless you change the image link in the <img> tag, otherwise your visitors will get an inactive link icon. The inactive link icon appears when the browser does not find the image in the required folder.
