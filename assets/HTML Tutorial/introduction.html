What do you need?

  • You don't need anything to learn HTML.
  • You don't need an HTML editor.
  • You don't need a server
  • You don't need a website
Edit HTML

HTML can be written and edited in many text editors such as Adobe Dreamweaver , Visua HTML l Studio or Notdap ++ .

However, in this tutorial you can use a simple HTML text editor such as Notepad to write and edit HTML tags. Rest assured that using a simple text editor is the best way to learn HTML.

Which extension is .HTML or .HTM ?

When we want to save an HTML file, you can save it with either the .html or .htm extension. Frankly, there is no difference between these two extensions. It is up to you.
