HTML Headings

HTML headings are defined by <h1> to <h6> tags

Examples


<h1>This is the title</h1>
<h2>This is the title</h2>
<h3>This is the title</h3>

HTML topic

The HTML subject is defined by the <p> tag

Example


<p>This is a topic</p>
<p>That's another topic</p>

HTML hyperlinks

HTML hyperlinks are defined by the <a> tag

Example


<a href="http://www.w3arabiconline.com">This is a hyperlink</a>

Note: The link address is identified by the suffix href

(you will learn more about suffixes in future lessons of this course)

Images in HTML

Images in HTML are defined by the <img> tag

Example


<img src="w3arabic.jpg" width="104" height="142" />

Note: Images can be named and sized using image extensions
