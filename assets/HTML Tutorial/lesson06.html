HTML Extensions

.HTML elements can contain extensions

.Suffixes provide additional information about the element

.Suffixes are always specified in the start tag

.Suffixes always come in pairs (noun/content) like this:

name = "value"

:Example of a suffix

HTML hyperlinks are identified by the <a> tag. The link address is identified by the href suffix.

Example:

<a href="http://www.example.com">This is a hyperlink</a>

The content of the appendices is always placed within quotation marks.

.The content of the suffix must always be enclosed in quotation marks. The double quotation mark is the most common, but single quotation marks can also be used.

Note: In some rare cases, when the content of the suffix itself includes a quotation, it is necessary to use single quotation marks.

Example:

name = 'John "ShotGun" Nelson'

Lowercase or uppercase for suffixes in HTML

.The names and contents of extensions are not case-sensitive. However, the World Wide Web Foundation recommends using lowercase extension contents in HTML4. Newer versions of (X)HTML will require lowercase extensions.
