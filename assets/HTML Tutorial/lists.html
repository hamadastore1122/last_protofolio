Lists

By mohammed yahya

The most popular HTML lists are the numbered list and the unnumbered (bulleted) list.

Numbered list

  1. The first element
  2. The second element
  3. The third element

Unnumbered list is dotted

A dotted list starts with the <ul> tag and each item starts with the <li> tag and ends with </li>. Items are identified by circles (usually small black circles).

Example

<ul>  
<li>Coffee</li>  
<li>Milk</li>  
</ul>

The menu code will appear in the browser as follows:

  • Coffee
  • Milk

Numbered list

A numbered list starts with the <ol> tag and each item starts with the <li> tag and ends with </li>. Items are identified by numbers.

Example

<ol>  
<li>Coffee</li>  
<li>Milk</li>  
</ol>

The menu code will appear in the browser as follows:

  1. Coffee
  2. Milk

Identification list

A definition list is made up of elements and each element is defined separately. The definition list starts with the <dl> tag and each defined element starts with the <dt> tag (which identifies the element in the list) in addition to <dd> which identifies the element in the list.

Example

<dl dir="rtl">  
<dt>Coffee</dt>  
<dd>Hot black drink</dd>  
<dt>Milk</dt>  
<dd>Cold white drink</dd>  
</dl>

The menu code will appear in the browser as follows:

Coffee
Hot black drink
Milk
Cold white drink
