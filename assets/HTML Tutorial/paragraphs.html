HTML Topics

Topics are identified by the <p> tag.

<p>This is a topic</p>

<p>This is another topic</p>

Note: Browsers automatically add a blank line before and after each topic.

Don't forget the end tag

Most browsers will display HTML correctly even if you forget the end tag.

<p>This is a topic</p>

<p>That's another topic</p>

The above example will work in most browsers, but you should not rely on it as omitting the end tag can cause errors and unexpected results.

Note: Future versions of HTML will not allow the omitting of the end tag.

New HTML line

Using the <br /> tag gives a new line without starting a new topic.

<p>This topic</p>

The <br /> element is an empty HTML element,

It has no end mark.

<br /> Mother <br />

In XML, XHTML elements without an ending tag are not allowed. Even though <br /> works in all browsers, writing <br /> instead works better in XHTML and XML applications.

HTML Output

You cannot be certain how HTML will appear. Large or small screens, and resized windows will produce different results. In HTML, you cannot change the output by adding extra spaces or lines to the HTML code. The browser will remove the extra spaces and whitespace when displaying the page.
