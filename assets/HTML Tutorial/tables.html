Tables

Tables are defined by the <table> tag. The table is divided into horizontal rows (using the <tr> tag) and each row contains cells that contain data (using the <td> tag). The <td> tag stands for table data, which holds the data content in the cell. The <td> cell can contain text, links, images, lists, forms, and other tables.

Example of a table

<table dir="rtl" border="1">  
<tr>  
<td>Row 1, Cell 1</td>  
<td>Row 1, Cell 2</td>  
</tr>  
<tr>  
<td>Row 2, Cell 1</td>  
<td>Row 2, Cell 2</td>  
</tr>  
</table>

The table will appear in the browser as follows:

Row 1, Cell 2 Row 1, Cell 1
Row 2, Cell 2 Row 2, Cell 1

Note: As long as we are using the Arabic language, we can always use the right-to-left direction suffix dir="rtl" in HTML or through the direction property direction: rtl; in CSS.

Table and suffix borders

If you do not specify the border extension, the table will appear without borders. Sometimes not specifying a border is useful, but be careful as most of the time we want to show the table borders. To display the table borders, only put the border extension with the value you want the border thickness to be.

<table border="1">  
<tr>  
<td>Row 1, Cell 1</td>  
<td>Row 1, Cell 2</td>  
</tr>  
</table>

Table header

The table header, i.e., the column headings, is specified by the <th> tag, and all browsers display the text inside the <th> tag as bold text and centered.

<table border="1">  
<tr>  
<th>Name</th>  
<th>Age</th>  
</tr>  
<tr>  
<td>Mohammed</td>  
<td>13</td>  
</tr>  
<tr>  
<td>Samer</td>  
<td>33</td>  
</tr>  
</table>

The previous code will appear as follows:

the age the name
13 Mohammed
33 Samer
