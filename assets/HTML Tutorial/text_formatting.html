Text Formatting Tags

Using HTML, you can use the tags <b> to make text bold and <i> to make text italic. These tags are called text formatting or editing tags.

Use <strong> instead of <b> and <em> instead of <i>.

In fact, <b> is used to make text bold and <i> to make text italic only, but although the <strong> tag shows the same result as the <b> tag and <em> shows the same result as <i>, there is a big difference in the nature and meaning of the tag for search engines.

When using <strong>, this means that the browser is telling the search engine that this text is strong and should be given importance, which is what makes the search engine prefer it over <b>. As for <em>, emphasis = focus - emphasis), when using this tag and making the text italic, I can tell the search engine that the text I used should be emphasized or emphasized, so the search engine will pay more attention to the text with this tag than <i>.
