Comparison symbols and logical symbols

By mohamd yahya | JavaScript

Comparison symbols and logical symbols are used to check whether the result is true or false.

Comparison symbols

Comparison symbols are used with logical conditions to determine similarity and difference between variables and values.

Let us assume that x=5 in the following table to explain comparison symbols:

example the explanation The symbol
x=8 is wrong is correct =
x=5 is True Exactly equal to (value and type) ===
x!=8 is True Not equal to !=
x>8 is wrong Bigger than >
x>=8 is wrong greater than or equal to >=
x<=8 is correct less than or equal to <=

How to use it:

Comparison symbols can be used with conditional expressions to compare values and cause something to happen based on the result.

<script type="text/javascript">  
var age = 18;  
if (age >= 18) {  
    document.write("Young man");  
}  
</script>

Conditional Symbols

JavaScript also contains conditional codes that assign a specific value to a variable based on certain conditions.

The conditional syntax is using from left to right:

variableName = (condition) ? value1 : value2;

For example:

<script type="text/javascript">  
var visitor = "PRES";  
var greeting = (visitor == "PRES") ? "Hello Mr. President" : "Hello dear";  
document.write(greeting);  
</script>
