What is JavaScript?

JavaScript was invented to add interactive HTML pages.

JavaScript is a scripting language.

Written language can be defined as a light programming language on the website pages.

JavaScript is usually embedded directly within HTML pages.

JavaScript is an instant language, meaning that the result of the code is displayed directly without any additional requirements or programs.

Anyone can use JavaScript without purchasing a license.

Is Java the same as JavaScript?

Never.

Java and JavaScript are two completely separate languages in design and concept.

Java (developed by Sun Microsystems) is a powerful and complex programming language similar in structure to C and C++.

What can JavaScript do?

JavaScript provides a programming tool for HTML pages: HTML page writers are not programmers, and although JavaScript is a programming language and with a simple structure, anyone can attach JavaScript codes to HTML pages.

JavaScript can be used with events: With this feature, events can be attached to HTML pages when something happens, such as the page loading finishes or when the user clicks on a specific HTML element on the web page.

JavaScript can write and read HTML elements: JavaScript can write and change the content of HTML elements.

JavaScript can be used with forms: JavaScript allows the form to be verified before sending it to the server in order to avoid additional operations being loaded on the server.

JavaScript can be used to identify the user’s browser: it can determine the type of browser the visitor is using and then open a specific page that is different depending on the browser he has.

JavaScript can be used to create browser cookies: data and information can be downloaded to the visitor’s computer, such as (browser history).

JavaScript = ECMAScript

JavaScript is an implementation of the ECMAScript standard. ECMA-262 is the official standard for JavaScript.

JavaScript was invented by Brendan Eich for Netscape but in 1996 JavaScript was supported in all browsers.

The official standard was adopted by ECMA (a trade standards organization) in 1997.

ECMA standards were approved by ISO International in 1998.

The language is still under development.
