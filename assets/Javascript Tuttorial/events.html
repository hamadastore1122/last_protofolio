Events

Using JavaScript you can create dynamic web pages. Events are actions that are created using JavaScript.

Every element in HTML can have events associated with it in JavaScript. For example, the onClick event on a button element can be used to indicate that a function code will be used when the user clicks the button. Events can be defined in HTML elements.

Examples of events:

  • When clicking with the mouse
  • When opening a page or image
  • When you hover your mouse over a certain point on a web page
  • When selecting a text field in HTML forms
  • When you click on the submit form button

Note: Events are usually used with function codes, meaning that the function codes will not run before the event occurs.

Using onLoad and onUnload events

The onLoad event is typically used to check the user's browser type and browser version and load the appropriate version for the web page on which the information relies.

Use onSubmit

The onSubmit event is used to activate all forms before they are submitted. In the following example we will learn how to use the onSubmit event.

<form method="post" action="xxx.htm" onsubmit="return checkForm()">

The checkForm() function will be called when the user clicks the "Submit" button to check the form. The checkForm() function will return either true or false. If the result is false, the form will be canceled.
