For...In Expression

By mohammed yahya | JavaScript

The For...In expression is used within the properties of a specific object.

The Composition

for (variable in object)  
{  
    Insert code here  
}

Note: Notice that the code in the for...in loop expression is shown once for each property.

Example

var person = {fname: "Ahmed", lname: "Al-Ahmed", age: 25};  
var x;  
for (x in person)  
{  
    document.write(person[x] + " ");  
}
