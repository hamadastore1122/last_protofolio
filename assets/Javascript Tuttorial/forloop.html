Loops using For

By mohammed yahya | JavaScript

Loops are used to create a set of codes that determine the number of times a number is entered or if a condition is true.

JavaScript Loops

When you write a specific code and you want this code to be similar several times by appearing the same again within a specific line:

Instead of writing the code more than once, you can use the code once and in successive loops, then the code you want appears more than once.

There are two types of loops in JavaScript:

  • Loops using for: By using a set of codes to specify a specific number of times.
  • Loops using while: by using a set of codes when a certain condition is met.

For Loop

You use for loops when you know in advance how many times your code should run.

The composition

for (variable=startvalue; variable<=endvalue; variable=variable+increment) {  
    // Code here  
}

Example

In the following example, a loop will be defined starting from i=0 and the code will continue to run as long as the variable i is less than or equal to 10 and will be incremented by one number 1 each time the loop is run.

Note: You can also use subtraction instead of addition and you can use any type of comparison from the greater and lesser signs and others that were explained in previous lessons.

<html>  
<body>  
<script type="text/javascript">  
var i = 0;  
for (i = 0; i <= 10; i++) {  
    document.write("The number is " + i + "
");  
}  
</script>  
</body>  
</html>
