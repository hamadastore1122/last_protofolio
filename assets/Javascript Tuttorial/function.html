Function code can be called upon a specific event or when the function code is called.

JavaScript Jobs

To avoid displaying codes in the browser when the page loads, you can put the code within a function code.

The function code that will appear in the browser through a specific event or through a call to the code. You can also call the code from anywhere in the web page or from another page where the code is located in a special JavaScript file with the extension .js.

How can a job code be defined?

The composition

Functional code can be used by first writing the word function, then the name of the function, then if we want to insert variables, we must put them between brackets, and finally between the curved brackets we put the code that we want to use.

function functionName(var1, var2, ..., varN) {  
    // Code here  
}

The parameters var1, var2, etc. are variables or values that are passed to the function code.

Note: When not using any parameters, you must use brackets () after the function code name.

Example

<html>  
<head>  
<script type="text/javascript">  
function displayMessage() {  
    alert("Welcome to the Arab Programming Institute!");  
}  
</script>  
</head>  
<body>  
<form>  
    <input type="button" value="Click here" onclick="displayMessage()" />  
</form>  
</body>  
</html>

Return Status

The return condition is used to specify a value that can be returned from the function code.

The following example shows the return of a value through two different numbers in a and b.

Example

<html>  
<head>  
<script type="text/javascript">  
function product(a, b) {  
    return a * b;  
}  
</script>  
</head>  
<body>  
<script type="text/javascript">  
document.write(product(8, 3));  
</script>  
</body>  
</html>

Lifetime of Variables in JavaScript

If you use a variable using or within a function, the variable will only be used within the function and will be ignored outside the function. These variables are called local variables.

If you use a variable outside the function, all code can use it and the variable's lifetime starts when it is created and ends when the page is closed.
