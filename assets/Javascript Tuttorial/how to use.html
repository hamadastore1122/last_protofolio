how to use

Writing code inside a web page

In the following example, JavaScript code will be used inside the <p> tag and the current date will be displayed inside it.

Example

  
              
            <html>  
            <body>  
            <h1>First JavaScript Page</h1>  
            <script type="text/javascript">  
            document.write("<p>" + Date() + "</p>");  
            </script>  
            </body>  
            </html>

Note: Try to avoid using document.write() function code in large JavaScript codes because the HTML page will overwrite code when using document.write() inside function code or when the page is loaded. However, writing document.write() function code is the easiest way to display the result of JavaScript code.

Change HTML elements

In the following example, JavaScript code will be used inside the <p> tag by calling a specific name for it and the current date will be displayed within it.

Example

  
              
            <html>  
            <body>  
            <h1>First JavaScript Page</h1>  
            <p id="demo">This is a paragraph.</p>  
            <script type="text/javascript">  
            document.getElementById("demo").innerHTML = Date();  
            </script>  
            </body>  
            </html>

Note: JavaScript can process HTML elements as it uses the DOM method to get the element through its ID getElementById(), i.e., through a unique name that can be used in JavaScript just like CSS.

Some browsers do not support JavaScript

If there are some browsers that do not support JavaScript, then it will cause a lot of problems.

To avoid these problems, you can use <!-- HTML comments --> to hide JavaScript code in case the browser does not support JavaScript. Just add the beginning of the comment code that starts with <!-- before writing JavaScript code and then add the end of the comment code --> after you finish writing the JavaScript code.

Example

  
              
            <html>  
            <body>  
            <script type="text/javascript">  
            // document.getElementById("demo").innerHTML = Date();  
            </script>  
            </body>  
            </html>
