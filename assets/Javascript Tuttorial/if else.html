If...Else Expressions

By mohammed yahya | JavaScript

Conditional expressions are used to create different commands based on different conditions.

Conditional expressions

When you write a specific code, you often want to create a specific command according to different conditions, and for this you can use the conditional expressions feature to do that, and we will learn how in the examples:

  • 1. If expression: This expression is used to display a specific code if the condition is completely true.
  • 2. If... else expression: This type of conditional code can be used if the first condition is true and the second is false.
  • 3. Expression if...else if...else: This expression is used with one of a set of codes, i.e. it is used when a certain condition is met. If it is not met, another condition will be displayed, and if it is not met, the code will be automatically applied when else.
  • 4. Switch expression: This expression is used to select a code from a set of codes.

If expression

This expression is used to display a specific code if the condition is absolutely true.

Syntax

if (condition) {  
    // The code that will appear if the condition is met  
}

Code explanation

We use the expression if, then between parentheses we put the condition we want to use, then between the curved brackets we insert the code we want to appear when this condition is met, and if the condition is not met, no result will be displayed.

Note: The word (if) must be written in lowercase letters, but when written in uppercase letters IF, JavaScript will display an error.

Example

<script type="text/javascript">  
/*  
The examination will be checked if the time is less than 10 it will appear  
Good morning message  
*/  
var d = new Date();  
var time=d.getHours();  
if (time<10) {  
    document.write("<strong>Good morning</strong>");  
}  
</script>

Note that we do not have the condition (else), meaning that the browser will be told to show the code only if the condition is met, meaning that if the time is after 10, no result will be shown.

if...else expression

If or Else Statement: This type of conditional code can be used if the first condition is true and the second is false.

Syntax

if (condition) {  
    // The code that will appear if the condition is met  
} else {  
    // The code that will appear if the condition is not met  
}

If...Else If...Else Expressions

Code explanation:

We use the expression if, then between parentheses we put the condition we want to use, then between the curved brackets we insert the code we want to appear when this condition is met, and if the condition is not met, the code inside else will be displayed, meaning if the condition is not met, the other code will appear.

Example

<script type="text/javascript">  
var d = new Date();  
var time = d.getHours();  
if (time < 10) {  
    document.write("<strong>Good morning</strong>");  
} else {  
    document.write("<strong>You are late today</strong>");  
}  
</script>

Expression if ... else if ... else

This expression is used with one of a set of codes, i.e. it is used when a certain condition is met. If it is not met, another condition will be displayed, and if it is not met, the code will be automatically applied when else.

The composition

if (condition 1) {  
    // The code that will appear if condition 1 is met  
} else if (condition 2) {  
    // The code that will appear if condition 2 is met  
} else {  
    // The code that will appear if condition 1 or condition 2 is not met  
}

Code explanation:

We use the expression if, then between parentheses we put condition 1 that we want to use, then between the curved brackets we insert the code that we want to appear when this condition 1 is met, then we put condition 2 with else if that we want to use, then between the curved brackets we insert the code that we want to appear when this condition 2 is met, and if the condition is not met, the code within else will be displayed, meaning the code will appear if condition 1 or condition 2 is not met.

Example

<script type="text/javascript">  
var d = new Date();  
var time = d.getHours();  
if (time < 10) {  
    document.write("<strong>Good morning</strong>");  
} else if (time >= 10 && time < 12) {  
    document.write("<b>It's a beautiful day</b>");  
} else {  
    document.write("<strong>You are late today</strong>");  
}  
</script>
