Add Words and Numbers

The rule says: If you add a word to a number, the result will be a word.

Note: When using any value between quotation marks, it is considered a word, i.e., x=5 here x contains the number 5, while x="5" in this example x contains the word 5, i.e., we considered the number 5 as a word.

Example

<script type="text/javascript">  
    /*  
    br tag added  
    To go down a line between each result and the next  
    */  
    x=5;  
    document.write(x + "<br />");  
    // The result is 10  
    x="5"+5;  
    document.write(x + "<br />");  
    // The result is 55  
    x=5+"5";  
    document.write(x + "<br />");  
    // The result is 55  
    x="5"+5;  
    document.write(x + "<br />");  
    // The result is 55  
</script>
