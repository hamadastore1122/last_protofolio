Alert Box

The alert box is used to inform you if there is something important for the user.

alert("Message: This is an alert box.");

Example

<html>  
<head>  
<script type="text/javascript">  
function show_alert() {  
    alert("Message: This is an alert box.");  
}  
</script>  
</head>  
<body>  
<input type="button" onclick="show_alert()" value="Click here to show alert" />  
</body>  
</html>

Confirm Box

Used for confirmation dialogs when you want to confirm something specific.

confirm("Are you sure?");

Example

<html>  
<head>  
<script type="text/javascript">  
function show_confirm() {  
    var r = confirm("Are you sure?");  
    if (r == true) {  
        alert("You pressed 'OK'");  
    } else {  
        alert("You pressed 'Cancel'");  
    }  
}  
</script>  
</head>  
<body>  
<input type="button" onclick="show_confirm()" value="Click here for confirmation" />  
</body>  
</html>

Prompt Box

Used to get a specific value from the user before entering the page.

prompt("Please enter your name:");

Example

<html>  
<head>  
<script type="text/javascript">  
function show_prompt() {  
    var name = prompt("Please enter your name:");  
    if (name != null) {  
        document.write("Hello " + name + "!");  
    }  
}  
</script>  
</head>  
<body>  
<input type="button" onclick="show_prompt()" value="Click here to enter name" />  
</body>  
</html>
