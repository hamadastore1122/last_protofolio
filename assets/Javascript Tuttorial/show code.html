Show JavaScript Code

By Ayman Mosully | JavaScript

JavaScript is a set of states that can be displayed on the browser.

JavaScript is case sensitive.

JavaScript is a sensitive sense of writing codes and it is not like HTML so be careful about using upper or lower case letters when writing or calling variables, functions and events.

Show JavaScript Code

To display JavaScript codes, you must write commands to the browser. The purpose of these commands is to tell the browser what to do.

In the following example, the browser will be ordered to write "Welcome to the Arab Programming Institute" on the web page:

document.write("Welcome to the Arab Programming Institute");

It is normal to add a semicolon ";" at the end of every code statement and programmers consider it an excellent programming practice to add a semicolon after every code statement and you will see that in JavaScript examples.

The semicolon is optional (according to JavaScript standards) and the browser can distinguish between the end of each line and each code statement but it is better to use the semicolon because you need to know where the line of code ends and where the code ends as well.

Note: The semicolon allows you to write more than one code statement on a single line.

JavaScript code

JavaScript code is a sequence of JavaScript commands and states. Each command appears on the browser in sequence as it is typed.

In the following example, a large title will be written with two paragraphs:

<script type="text/javascript">  
document.write("<h1>This is a big title</h1>");  
document.write("<p>This is a paragraph</p>");  
document.write("<p>This is another paragraph</p>");  
</script>

JavaScript groups

JavaScript codes can be grouped together as blocks. Groups start with curly braces "{" and end with "}".

The purpose of using groups is to sequence codes according to their commands with each other.

In the following example, a large title will be written with two paragraphs:

<script type="text/javascript">  
{  
document.write("<h1>This is a big title</h1>");  
document.write("<p>This is a paragraph</p>");  
document.write("<p>This is another paragraph</p>");  
}  
</script>

The above example is not very useful when using codes as groups but only to show how to use braces in groups. Groups are generally used with function codes or in conditions (where a set of commands will be displayed if a condition is met).

You will learn more about function codes and conditions in future lessons.
