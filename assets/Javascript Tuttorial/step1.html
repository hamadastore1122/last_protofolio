First step with JavaScript

By mohammed yahya

JavaScript is a programming language developed by Netscape and Sun Microsystems. JavaScript differs from Sun Microsystems' Java language and the similarity in name is not because they are the same. The benefit of this language is to breathe life into the Internet, as it is used to create more interactive (dynamic) web pages.

There are two patterns of JavaScript in terms of implementation on the client and on the server. The first is to write code with the HTML page and then the operations requested by the user are executed on his device, i.e., within his browser, and the second forces the browser to communicate with the server in order to execute the commands and give the result to the client to display. The first has the disadvantage of slow page loading for the first time and then the speed of execution during the session, but the second is characterized by the speed of loading for the first time and the slow response due to the communication process with the server for each request from the client.

Interest in JavaScript has increased, especially after the spread of AJAX technology, which led to speed in interacting with the user.

This language was created to work entirely through the browser, so it cannot be used to produce programs that work alone without the browser, and in this it is like other network languages such as HTML and others.

Recently, with the launch of Windows 8, developers have been able to program Metro Style App applications using JavaScript and HTML5 by integrating them with the new WinRT API, but they do not run on web browsers and work as standalone applications under Windows 8 and compatible devices.
