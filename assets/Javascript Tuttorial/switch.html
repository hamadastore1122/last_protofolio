Switch Expression

By Ayman Mosully | JavaScript

Conditional expressions are used to create different commands based on different conditions.

Switch Expression in JavaScript

The switch expression is used to select a code from a set of codes.

The Composition

switch(n) {  
    case 1:  
        Insert code set 1;  
        break;  
    case 2:  
        Insert code set 2;  
        break;  
    default:  
        Show the code if the result does not match cases 1 and 2;  
}

How does the code work?

First, when using the switch expression, you must specify a specific expression between parentheses. It is usually a variable and is evaluated once. Then the value of the expression to be compared is compared with the other values in each case.

If there is a match with the code, it will be displayed immediately and the code will stop checking. You can also use break after each case to avoid the check passing to the other case automatically.

Example

<script type="text/javascript">  
/*  
You will receive different greetings depending on what day it is today.  
*/  
var d = new Date();  
var theDay = d.getDay();  

switch (theDay) {  
    case 5:  
        document.write("Finally Friday");  
        break;  
    case 6:  
        document.write("Saturday is the start of the work day");  
        break;  
    case 0:  
        document.write("Sunday is another day of work");  
        break;  
    default:  
        document.write("I am waiting for the holiday");  
}  
</script>

Note: We use the default case when the conditions are not met in each of the previous cases to display a separate message, exactly as else works.
