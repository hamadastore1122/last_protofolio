Checking Errors in JavaScript

The try...catch statement allows you to test a block of code for errors. It is used to handle exceptions that may occur during the execution of code.

Try...Catch Structure

try {  
    // Code to be tested for errors  
} catch (err) {  
    // Code to handle the error  
}

Example

<html>  
<head>  
<script type="text/javascript">  
var txt = "";  
function message() {  
    try {  
        addError();  
    } catch (err) {  
        txt = "An error occurred: " + err.message;  
    }  
    alert(txt);  
}  
</script>  
</head>  
<body>  
<input type="button" value="Try it" onclick="message()">  
</body>  
</html>

Another Example

<html>  
<head>  
<script type="text/javascript">  
var txt = "";  
function message() {  
    try {  
        txt = prompt("Please enter a number:", "");  
        if (txt == null || txt == "") {  
            throw "User cancelled the prompt.";  
        }  
    } catch (err) {  
        alert("An error occurred: " + err);  
    }  
}  
</script>  
</head>  
<body>  
<input type="button" value="Try it" onclick="message()">  
</body>  
</html>
