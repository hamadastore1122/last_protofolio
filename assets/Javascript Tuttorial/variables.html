JavaScript Variables

Like algebra, JavaScript is used to collect values and expressions. Variables can be the letter x or a word such as carname.

JavaScript Variable Naming Rules

The variable name is case-sensitive, i.e., x is different from X. Variable names must begin with a letter, a $ sign, or an _ sign. Variable names cannot be written in Arabic only. They can be used with expressions in any language.

Note: Because JavaScript is case sensitive, variables are also case sensitive.

Create JavaScript Variables

Creating variables in JavaScript often means declaring variables. You can declare a variable by putting the word var before the variable name.

Example

var x;  
var carname;

After creating the above code, you will need to add values to make the variables effective. Values can be placed in the variable and they can also be displayed.

Example

var x = 5;  
var carname = "Mazda";

After specifying the values for each of the variables, we specified the value 5 for the variable x and the value Mazda for the car name carname.

Note: When using text as a value in the variable, you must use quotation marks around the value. The variable will not lose its value if it is not called.

Local JavaScript Variables

When a variable is used within a function, the variable is called a local variable, meaning it can only be used within the function code.

Local variables can only be used within function codes because local variables operate within function codes even if they are called from other specific functions. Local variables are canceled when you exit the function code.

External JavaScript Variables Global

When a variable is used outside the functional code, it is called an external variable, meaning that these variables can be called with all functional codes on the site.

External variables are canceled when the page is closed. When the word var is not used before the variable name, the variable will become external.

Setting Values When JavaScript Variables Are Not Called

When you specify a value without calling it, the variable will be called automatically as if it were an external variable.

x = 5;  
carname = "Volvo";

When these expressions x and carname are used without the var keyword they will become external variables (unless they are used before).
