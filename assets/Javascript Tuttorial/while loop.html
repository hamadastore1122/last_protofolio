While Loop

By Ayman Mosully | JavaScript

Loops are used using while statements by using a set of codes when a certain condition is met.

The composition

while (variable <= endvalue) {  
    // Code here  
}

Note: You can use any type of comparison from the greater and lesser signs and others that were explained in previous lessons.

Example

In the following example, a loop will be defined starting from i=0 and the code will continue to run as long as the variable i is less than or equal to 5 and will be incremented by one number 1 each time the loop is repeated.

<html>  
<body>  
<script type="text/javascript">  
var i = 0;  
while (i <= 5) {  
    document.write("The number is " + i);  
    document.write("<br />");  
    i++;  
}  
</script>  
</body>  
</html>

Loops using do...while

do...while is a type of while loop and is used to display a set of codes once and then the loop is repeated as long as the specified condition is true.

The composition

do {  
    // Code to be displayed  
} while (variable <= endvalue);

Example

In the following example, a do...while loop will be used and the code will be shown once even if the condition is false because the expression was created before the condition was checked.

<html>  
<body>  
<script type="text/javascript">  
var i = 0;  
do {  
    document.write("The number is " + i);  
    document.write("<br />");  
    i++;  
} while (i <= 10);  
</script>  
</body>  
</html>
