PHP Language: If...Else Expressions

By mohammed yahya | PHP

Conditional expressions are used to create different commands based on different conditions.

Conditional expressions

When you write a specific code, you often want to create a specific command according to different conditions, and for this you can use the conditional expressions feature to do that, and we will learn how in the examples.

We have 4 conditional expressions, which are:

  • If expression: This expression is used to display a specific code if the condition is completely true.
  • If... else expression: This type of conditional code can be used if the first condition is true and the second is false.
  • Expression if...else if...else: This expression is used with one of a set of codes, i.e. it is used when a certain condition is met. If it is not met, another condition will be displayed, and if it is not met, the code will be automatically applied when else.
  • Switch expression: This expression is used to select a code from a set of codes.

If expression

This expression is used to display a specific code if the condition is absolutely true.

The composition

if (condition) {  
    // The code that will be displayed if the result is true;  
}

The following example will display "Today is Friday" if today is Friday:

<html>  
<body>  
<?php  
$d=date("D");  
if ($d=="Fri") echo "Today is Friday";  
?>  
</body>  
</html>

Note that we did not use the else expression in this example. The result of the code will only be displayed when the result is true.

if...else expression

If or Else Expression: This type of conditional code can be used if the first condition is true and the second is false.

The composition

if (condition) {  
    // The code that will appear if the condition is met  
} else {  
    // The code that will appear if the condition is not met  
}

PHP Conditional Expressions

Example

The following example will display "Today is Friday" if today is Friday. If it is not Friday, it will display the result: "Today is not Friday".

<html>  
<body>  
$d=date("D");  
if ($d=="Fri") {  
    echo "Today is Friday";  
} else {  
    echo "Today is not Friday";  
}  
</body>  
</html>

Multiple Results

If there is more than one result to be displayed if the condition is true/false, then the condition and the results must be enclosed in curly braces {}.

<html>  
<body>  
$d=date("D");  
if ($d=="Fri") {  
    echo "Hello!
";  
    echo "It's a beautiful day!";  
    echo "See you later!";  
}  
</body>  
</html>

Expression if ... else if ... else

This expression is used with one of a set of codes, i.e. it is used when a certain condition is met. If it is not met, another condition will be displayed, and if it is not met, the code will be automatically applied when else.

The composition

if (condition 1) {  
    // The code that will appear if condition 1 is met  
} else if (condition 2) {  
    // The code that will appear if condition 2 is met  
} else {  
    // The code that will appear if condition 1 or condition 2 is not met  
}

Example

In the following example, the sentence "Have a nice weekend!" will be displayed if today is Friday, otherwise "Have a nice Sunday!" will be displayed if it is Sunday, and "Have a nice day!" will be displayed otherwise.

<html>  
<body>  
$d=date("D");  
if ($d=="Fri") {  
    echo "Have a nice weekend!";  
} elseif ($d=="Sun") {  
    echo "Have a nice Sunday!";  
} else {  
    echo "Have a nice day!";  
}  
</body>  
</html>
