PHP Installation

By mohammed yahya | PHP

What do you need?

If your server supports PHP then you don't need to install or do anything.

You can just create PHP files in your server's folder and the codes will be converted, read, and displayed on the browser.

However, if your browser does not support PHP then you must install PHP on the server.

From here you can go to the official PHP website with a full explanation of how to install PHP5:

http://www.php.net/manual/en/install.php

Installing PHP on the server is not an easy matter and requires a number of programmers to install it, so we will not delve into how to install it, but rather into how to create a website and pages in PHP.

Download PHP

To download PHP for free, click on the link: http://www.php.net/downloads.php

Download MySQL databases

To download MySQL, click on the link: http://www.mysql.com/downloads

Download Apache Server

To download the Apache server through the link: http://httpd.apache.org/download.cgi
