Text variables are used to store and modify texts.

String variables in PHP

String variables are used for values that contain letters and words. In this lesson, we will look at the most common functions and operators used as strings in PHP. We can modify the text in the variable after it is created, and it can be called directly or saved and modified later.

In the following example, a variable containing a string value was created and called to display the result in the browser:

<?php  
$text = "Welcome";  
echo $text;  
?>

The result will appear on the browser:

Welcome

Linking codes

There is one link used with PHP to link codes together and it is . and it is used to link two text values together.

Example:

<?php  
$text1 = "Welcome to ";  
$text2 = "Arab Institute for Programming";  
echo $text1 . " " . $text2;  
?>

The result will appear:

Welcome to Arab Institute for Programming

Function code strlen()

The strlen() function is used to determine the length of the text, i.e., the number of characters of the text used. The strlen() function code is usually used with loops or some other function code especially when it is important to know when the loop will stop.

Example:

<?php  
echo strlen("Hello world!");  
?>

The result will appear on the browser:

12

Function code strpos()

The strpos() function is used to search for text or characters within a single text. If the result is found, the location of the word or character will be displayed or matching the result. However, if no result is found, the result will be FALSE.

Now we will search for the word "world" within the text:

<?php  
echo strpos("Hello world!", "world");  
?>

The result will appear on the browser:

6

The position of the word in the text is 6 in the previous example. The reason is 6 and 7 is that the letter of the text begins with the number 0 and not 1.
