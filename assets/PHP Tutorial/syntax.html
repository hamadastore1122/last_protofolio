PHP codes run on the server and the result appears on the browser in the form of HTML texts.

Main Configuration

PHP codes always start with the <?php tag and end with ?>. You can also insert PHP codes anywhere you want on your web page.

Some servers that have shortcuts for PHP codes can start with the <? tag and end with ?>. However, we strongly recommend using the official code tag to avoid server problems, which is <?php.

<?php  
?>

A PHP file contains HTML tags just like HTML pages but also contains special PHP code.

An example of a web page containing simple PHP code and the text "Welcome" that will appear in the browser:

<html>  
<body>  
<?php  
echo "Welcome";  
?>  
</body>  
</html>

At the end of each PHP code, there must be a semicolon; this semicolon is the mark that separates the codes for easy reading.

The code can be displayed in the browser through expressions used in PHP such as echo and print.

Note: The file must be PHP and its extension is .php. If the PHP code is in .html pages, the code will not work.

Comments and Notes in PHP

In PHP we use // for in-line comments and we can also use /* and */ for multi-line comments just like in JavaScript.

<html>  
<body>  
<?php  
// one line comment  
/* Comment with more than  
line */  
?>  
</body>  
</html>
