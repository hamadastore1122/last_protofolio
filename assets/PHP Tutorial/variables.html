A variable is used to store information.

Variables in PHP

Variables are used to store values such as strings, numbers, and arrays. Once created, a variable can be used anywhere and multiple times in a web page. All PHP variables start with $.

The best and most correct way to create a variable is as follows:

$var_name = value;

Some new programmers forget to put $ sign before the variable name and then the code will not work.

Now we will create two variables, one contains text and the other contains a number.

<?php  
$text = "Hello World!";  
$x = 16;  
?>

PHP is lax

Using PHP you don’t need to declare the variable before adding the value to it. In the previous example you don’t need to define the variable whether the data type is text, numbers, or other, because PHP automatically determines the data type of the variable once you use the value attached to the variable.

In strict programming languages you must determine the type of the variable before adding data to it, while PHP is considered a tolerant and lenient language, meaning the type of the variable is determined when the value is added to it.

Rules for choosing a variable name

The variable name must start with a lowercase letter or an underscore (_).

The variable name must contain lowercase letters or numbers (a-z, A-Z, 0-9, _).

The variable name must not contain spaces between names and words can be separated using _ (e.g., my_string) or by capitalizing the first letter of each word (e.g., myString).
