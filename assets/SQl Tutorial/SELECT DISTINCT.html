SELECT DISTINCT

By Ayman Mosully

The SELECT DISTINCT expression will be explained.

SELECT DISTINCT

Some columns in the table contain duplicate values, but this is not a big problem as you can only select the different values in the column by using DISTINCT, which is used to return only different values.

The composition

SELECT DISTINCT column_name(s) FROM table_name

Example:

City Address FirstName LastName P_Id
Sandnes Timoteivn 10 Ola Hansen 1
Sandnes Borgyn 23 Tove Svensdon 2
Stavanger 20th of July Kari Pettersen 3

Now the different values from the City column will be selected through the SELECT DISTINCT expression:

SELECT DISTINCT City FROM Persons

The result will appear as follows:

City
Sandnes
Stavanger
