What is SQL language?

  • SQL stands for Structured Command Language.
  • SQL allows you to access and modify the database.
  • SQL language is ANSI, which is American National Standards Institute.

What can SQL do?

  • Can execute commands related to databases.
  • Data can be retrieved from databases.
  • Table rows can be inserted into databases.
  • Rows of tables in databases can be updated.
  • Rows of tables in databases can be deleted.
  • A new database can be created.
  • New tables can be created in the database.
  • Data can be stored in a database.
  • Views can be created for databases.
  • Permissions can be set on tables, views, and databases.

SQL is standard but...

Although SQL is the standard language for databases, there are many different versions of it. Despite the differences in versions, most of them use the same standard commands such as SELECT, UPDATE, DELETE, INSERT, WHERE.

Note: Most database programs use their own programming and commands for databases in addition to SQL.

Using SQL with your own website

To create a site that uses content through databases, you need the following:

  • RDBMS database software such as MySQL, SQL Server, MS Access.
  • A server-side programming language such as PHP or ASP.
  • SQL language.
  • HTML/CSS.

What is RDBMS?

RDBMS stands for Relational Database Management System. RDBMS is used with all database languages including MS SQL Server, IBM DB2, Oracle, MySQL, Microsoft Access.

A table is a collection of input data and contains columns and rows.
