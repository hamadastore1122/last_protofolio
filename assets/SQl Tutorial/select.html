SELECT Statement

The SELECT statement is used to select data from a database.

Statement Structure

SELECT column_name(s) FROM table_name
SELECT * FROM table_name

Note: SELECT is case-sensitive in SQL statements, meaning SELECT is different from select.

Example:

Table of Persons:

City Address FirstName LastName P_Id
Sandnes Timoteivn 10 Ola Hansen 1
Sandnes Borgyn 23 Tove Svensdon 2
Stavanger Storgt 20 Kari Pettersen 3

Now let's extract the contents of the columns named FirstName and LastName:

SELECT LastName, FirstName FROM Persons

The result will be as follows:

FirstName LastName
Ola Hansen
Tove Svensdon
Kari Pettersen

Example of SELECT

Now we will retrieve all columns available in the table using the SELECT statement:

SELECT * FROM Persons

Note: Use the asterisk (*) to select all columns in the table.

The result will be displayed as follows:

City Address FirstName LastName P_Id
Sandnes Timoteivn 10 Ola Hansen 1
Sandnes Borgyn 23 Tove Svensdon 2
Stavanger Storgt 20 Kari Pettersen 3

This demonstrates how most programming languages interact with databases to retrieve data, showing the result on the interface, such as PHP or ADO.
